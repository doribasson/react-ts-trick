הכל נמצא ב execution context
במקום אחד שם נמצא הזיכרוןמששומר משתנים כאנדפיינד בהתחלה ויש מקום ששם מריץ את הקוד.
קודם כל כל המשתנים של ה var נשמרים במקום שנקרא גלובל  ונכנסים כ אנדיפיינד.
ואז שמסתיימים כל המשתנים הפונקציה מתחילה לרוץ שורה שורה בקוד.  ושמופעל פונקציה שוב נוצר חלון של ממורי ושוב של קוד כמו מקודם.
עד שכל הקוד מסתיים.
יש לנו את הסטק שכל פעם מריץ משהו אחד ושמסיים מוציא מהסטק וממשיך שוב לשורת קוד הבאה שזה למשל הפונקציה וברגע שהכל מסתיים אז הכל מסתיים בסטק.
